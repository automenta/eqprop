# EqProp-Torch Tests

This directory contains unit tests for the eqprop-torch library.

## Running Tests

```bash
# Run all tests
python -m pytest eqprop_torch/tests/

# Run specific test file
python eqprop_torch/tests/test_library.py
python eqprop_torch/tests/test_bioplausible.py
```

## Test Coverage

- **test_library.py**: Core library tests (models, trainer, utils)
- **test_bioplausible.py**: Research algorithm wrapper tests
